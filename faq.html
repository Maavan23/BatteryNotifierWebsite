Frequently Asked Questions

What is Battery Notifier?

Battery Notifier is a lightweight desktop application that delivers desktop notifications and optional sound alerts when your battery reaches configurable low or full thresholds. It runs in the system tray and can optionally auto-start on Windows sign-in.

What's new in v1.6?

v1.6 improves application stability and runtime reliability, fixes asset path resolution for bundled resources (battery_icon.ico, short_bell.mp3), ensures user data (settings.json, history.json) is stored persistently in %APPDATA%\BatteryNotifier, and resolves startup conflicts with PyInstaller frozen executables.

Features

Desktop notifications for low/full battery, configurable thresholds, sound alarm with repeat limiting, notification history viewer, silent system tray mode, and optional auto-start on Windows boot.

Installation

Download the installer (e.g., BatteryNotifier_v1_6_Setup.exe) from the Releases page, run the installer, and follow the on-screen steps. The app will appear in your system tray after installation.

System Requirements

Windows 10 / 11 (64-bit). No additional software required — the installer is self-contained.

Where are my settings stored?

User data such as settings.json and history.json are stored in %APPDATA%\BatteryNotifier so they survive reinstalls and updates.

How can I download the latest release?

The download button on the main site links to the latest release. You can also visit the project's releases: Releases.
